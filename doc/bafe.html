项目说明

这个项目是我在玩JSLinux的时候，无意中发现一个叫red的命令，然后我就执行了下，发现一回车就是一问号，直接查了下AI，发现这个命令其实是一个行编辑器，是ed的受限版本。又转念一想，我自己也能用bash开发一个行编辑器呀，然后就有了这个编辑器。

BAFE是一个bash脚本，全称是BAsh File Editor BASH文件编辑器。

快速开始

快速开始，也就是安装说明(?)

1.安装git

这个应该都会吧，我就不讲了。

(不讲的原因，第一懒得讲应该都会，第二，每个系统的包管理器都不一样)

2.克隆仓库

git clone https://github.com/chenwumm/bafe/

3.愉快的(?)使用BAFE

在你的命令行执行

bash ~/bafe/bafe.sh

这个是中文版，如果你的终端不支持显示中文字符建议执行英文版。

英文版执行

bash ~/bafe/bafe-en.sh

使用帮助

命令 说明
i 进入插入模式（输入 '.' 结束）
p 打印当前行
a 在末尾追加文本
d 删除当前行
w 文件名 保存文件
l 文件名 加载文件
/关键词 搜索内容
n 跳转到下一个匹配项
q 退出编辑器
h 显示帮助
数字 跳转到指定行（如 '3'）
