Project Description

This project was inspired when I was playing with JSLinux and accidentally discovered a command called 'red'. After executing it, I saw it just showed a question mark. After checking with AI, I learned it was actually a line editor - a restricted version of 'ed'. Then it occurred to me that I could develop my own line editor using bash, and thus this editor was born.

BAFE is a bash script, with the full name being BAsh File Editor.

Quick Start

Quick start, aka installation guide (?)

1. Install git

You should already know how to do this, so I won't explain it.

(The reasons I'm not explaining: 1. You should know already, 2. Package managers differ across systems)

2. Clone the repository

git clone https://github.com/chenwumm/bafe/

3. Enjoy (?) using BAFE

Execute in your command line:

bash ~/bafe/bafe.sh

This is the Chinese version. If your terminal doesn't support Chinese characters, it's recommended to use the English version.

For English version execute:

bash ~/bafe/bafe-en.sh

Usage Guide

Command Description
i Enter insert mode (enter '.' to end)
p Print current line
a Append text at the end
d Delete current line
w filename Save file
l filename Load file
/keyword Search content
n Jump to next match
q Quit editor
h Show help
number Jump to specified line (e.g. '3')
