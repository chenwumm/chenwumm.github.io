Project Description

This project began when I was experimenting with JSLinux and accidentally discovered a command called 'red'. After executing it, I noticed it displayed a question mark upon pressing enter. After consulting AI, I learned it was actually a line editor - a restricted version of 'ed'. Then it occurred to me that I could develop my own line editor using bash, and thus BALE was born.

BALE is a bash script, with the full name "BAsh File Editor".

Quick Start

Quick start, aka installation instructions (?)

1. Install git

You probably know how to do this already, so I won't explain.

(The reasons I'm not explaining: first, you should already know; second, package managers vary across different systems)

2. Clone the repository

git clone https://github.com/chenwumm/bale/

3. Enjoy (?) using BALE

Execute in your command line:

bash ~/bale/bale.sh

This is the Chinese version. If your terminal doesn't support Chinese characters, we recommend using the English version.

For English version execute:

bash ~/bale/bale-en.sh

Usage Help

?BALE Line Editor - Help Manual

  • i Enter insert mode (insert before current line, enter '.' to end)
  • I Insert at beginning of file (enter '.' to end)
  • a Append after current position (enter '.' to end)
  • A Append at end of file (enter '.' to end)
  • p Print current line
  • P Print all lines (with line numbers)
  • n Show total line count
  • d Delete current line
  • D Delete all lines
  • w [file] Save file
  • l [file] Load file
  • /pattern Search content
  • N Jump to next match
  • g line# Jump to specific line
  • q Quit editor (confirm unsaved changes)
  • Q Force quit without saving
  • h Show help
  • !command Execute shell command
